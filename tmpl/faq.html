1. 己思的内容源是否都有版权？

我们对于版权是十分重视的，目前的 RSS 源主要有三类：
  1. 博主有声明可自由转载的，即 知识共享-非商用-非衍生-保持署名 协议
  2. 没有转载声明的，我们会尽量和博主取得联系
  3. 未知版权的，我们会以摘要转载的形式展现（即只展示摘要部分）
需要注意的是，有部分 RSS 源是用户提交的，如果发现其中有侵犯了您的权益，欢迎留言或邮件告知（ richshaw#126.com #替换为@），我们会在24小时内处理

2. 己思对于博主来说是否减少了访问量？

己思可以看作是另一种形态（在线）的 RSS 阅读器，只保留最近一周的内容，并不会对源站点造成冲击。 事实上反而给不少站点增加了流量，因此两者是共生关系，而不是竞争关系

3. 帐号是怎么管理的？

  • 所有的操作都可以以游客身份完成
  • 支持三方平台的 OAuth 登录认证，目前仅支持 GitHub
登录后会享受更多特权，如果您有 GitHub 账号的话推荐登录后使用

4. 我提交的 RSS 源为什么没有更新？

用户提交的 RSS 源一周后会被重新评分，为了充分利用计算资源，评分较低的源如果连续3天没有用户访问， 就会停止更新，直到有用户请求触发恢复更新。

PS：高评分源的典型特征：

  • IT 领域
  • 有技术深度或价值
  • 原创内容

5. RSS 源的同步周期是多久？

普通源一般是每 1~2 小时同步一次，像周报、月报这种内容一般会在周末或者月初特定的时间节点进行同步

6. 移动版网站有些按钮按不了？页面跳转不正常？

这是因为手机端的浏览器兼容性不同，建议使用手机版 Chrome 浏览器打开
为了确保最佳的用户体验，不建议在其他 APP （例如微信、QQ、微博）内打开

7. 不能看到历史文章？

目前的设定是只展示当前用户订阅的一周内的文章，历史内容只能通过搜索获得
所以，请及时消费您订阅的内容

8. 登录后有什么特权？

登录后可以跨平台同步数据、查看更多订阅数据、更好的用户体验等
后续会有更多特权推出，推荐登录使用

9. 如何添加公众号订阅？

目前支持二十次幂上的微信公众号
在「我的订阅」里提交对应的公众号主页即可，以「微信开发者」公众号为例，提交：https://www.ershicimi.com/a/5558

10. 我有业余时间，如何参与到这个项目中来？

己思有多名维护者，都是线上协作。有多种参与渠道：
  • 微信交流群（添加管理员 richshaw，备注：己思）
  • Slack 需求讨论
  • GitHub
在内容源建设、需求、营销推广等方面，我们都非常欢迎志愿者参与进来。如果表现出色，更有机会成为核心维护组成员~
